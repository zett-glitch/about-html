Cara Membuat Web HTML

Untuk membuat website menggunakan HTML, kamu hanya membutuhkan text editor (seperti Notepad atau VS Code) dan browser (Chrome, Firefox, dan lainnya).

Struktur Dasar HTML

1. <!DOCTYPE html>
Menentukan tipe dokumen HTML5.

2. <html>
Elemen utama yang membungkus seluruh isi website.

3. <head>
Berisi informasi halaman seperti judul, charset, dan style.

4. <body>
Berisi konten yang tampil di browser.
